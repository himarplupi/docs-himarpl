Manajemen konfigurasi dan rahasia yang tepat sangat penting untuk keamanan dan pemeliharaan. Setiap proyek memiliki pengaturan dan variabel lingkungan sendiri yang dapat Anda ikuti langkah-langkahnya di panduan kontribusi masing-masing proyek atau halaman dokumentasi ini.

## Environment Variables

:::note
**Catatan Penting untuk Staf Kominfo**

Semua Environment Variable disimpan di [Google Drive utama HIMARPL](https://drive.google.com).
Terdapat di folder `/KABINET ASCENDIA 2024/BADAN EKSEKUTIF/DEPT. KOMINFO/Data/Environment Variables`.
:::

Untuk setiap proyek, terdapat file `.env.example`. Anda dapat menggunakan ini sebagai template untuk membuat file `.env` untuk Environment lokal Anda.
