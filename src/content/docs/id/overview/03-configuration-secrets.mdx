Manajemen konfigurasi dan rahasia yang tepat sangat penting untuk keamanan dan pemeliharaan. Setiap proyek memiliki pengaturan dan _Environment Variable_ sendiri yang dapat Anda ikuti langkah-langkahnya di panduan kontribusi masing-masing proyek atau halaman dokumentasi ini.

## Environment Variables

:::note
**Catatan Penting untuk Staf Kominfo**

Semua _Environment Variable_ disimpan di [Google Drive utama HIMARPL](https://drive.google.com).
Terdapat di folder `/KABINET ASCENDIA 2024/BADAN EKSEKUTIF/DEPT. KOMINFO/Data/Environment Variables`.
:::

Untuk setiap proyek, terdapat _file_ `.env.example`. Anda dapat menggunakan ini sebagai _template_ untuk membuat _file_ `.env` untuk _Local Environment_ Anda.
