[blog.himarpl.com](https://blog.himarpl.com) adalah platform blog resmi untuk anggota HIMARPL (Himpunan Mahasiswa Rekayasa Perangkat Lunak). Situs web ini dibangun menggunakan Next.js dengan T3 Stack dan dilengkapi dengan editor teks keren yang didukung oleh TipTap.

## Fitur Utama

- Editor Teks Keren dengan Antarmuka WYSIWYG
- Autentikasi & Otorisasi Pengguna
- Penyusunan Artikel Berbasis Tag
- Pemuatan Artikel _Infinite Scroll_
- Fungsionalitas Pencarian
- Desain Responsif
- Mendukung Tema Gelap/Terang
- Berbagi antar Media Sosial
- Integrasi Analitik
