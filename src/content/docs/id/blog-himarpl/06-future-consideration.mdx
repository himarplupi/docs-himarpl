Mari kita bicara tentang beberapa hal yang perlu dipertimbangkan oleh kabinet HIMARPL mendatang ketika mereka bekerja untuk meningkatkan situs Blog HIMARPL.

## Apa yang Kami Perhatikan

- Saat ini, staf HIMARPL tidak terlalu banyak menggunakan situs web. Ini sebenarnya bukan kesalahan tim kominfo (kami hanya membangun situs web!) - lebih tentang membuat orang sadar dan bersemangat untuk menggunakannya. Kabinet berikutnya harus fokus untuk melibatkan lebih banyak orang.

- Kode pasti perlu dirapihkan. Kami harus membangun situs web ini dengan cukup cepat untuk memenuhi tenggat waktu, sehingga agak berantakan. Tim mendatang mungkin ingin merapihkan dan mengatur ulang kode untuk membuatnya lebih mudah untuk dikerjakan dan dikembangkan.

## Beberapa Ide yang Perlu Dipertimbangkan

- Cobalah menggunakan PostHog untuk melihat bagaimana orang sebenarnya menggunakan situs web - ini dapat membantu mengetahui apa yang bekerja dan apa yang perlu diperbaiki.

- Luangkan waktu untuk merapihkan kode, terutama di _Library_ komponen Editor. Ini akan membuatnya jauh lebih mudah bagi tim mendatang untuk menambahkan fitur baru!
