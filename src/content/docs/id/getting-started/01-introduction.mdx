Selamat datang di dokumentasi resmi proyek open-source HIMARPL! Dokumentasi ini berfungsi sebagai panduan komprehensif untuk semua proyek HIMARPL UPI (Himpunan Mahasiswa Rekayasa Perangkat Lunak Universitas Pendidikan Indonesia) dan sumber daya teknisnya.

## Apa itu HIMARPL?

HIMARPL (Himpunan Mahasiswa Rekayasa Perangkat Lunak) adalah Himpunan Mahasiswa Rekayasa Perangkat Lunak di Universitas Pendidikan Indonesia. Kami mengelola beberapa proyek open-source yang mendukung infrastruktur digital organisasi kami, mulai dari website utama hingga platform blog seperti medium untuk staff dan kolaborator kami.

## Mengapa Dokumentasi Ini Ada?

1. **Pusat Pengetahuan**

   - Sumber tunggal kebenaran untuk semua dokumentasi teknis
   - Praktik pengembangan yang terstandarisasi di semua proyek
   - Panduan kontribusi yang jelas

2. **Onboarding & Pembelajaran**

   - Onboarding yang mudah (semoga) untuk kontributor baru
   - Sumber pembelajaran untuk mahasiswa RPL
   - Dokumentasi struktur dan arsitektur proyek

3. **Keberlanjutan Proyek**
   - Praktik terbaik dan standar yang terdokumentasi
   - Panduan pemeliharaan untuk tim mendatang
   - Pelacakan dan pengelolaan technical debt

## Tech Stack Kami

Proyek HIMARPL terutama menggunakan teknologi web modern:

- **Fullstack**: Next.js, Tailwind CSS, Prisma, T3 Stack
- **Dokumentasi**: Astro, Starlight
- **Deployment**: Vercel, GitHub Actions

Untuk informasi lebih detail, silakan lihat bagian [tech stack](/id/overview/02-tech-stack).
