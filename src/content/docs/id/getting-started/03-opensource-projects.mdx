HIMARPL memelihara beberapa proyek _open-source_ yang dapat dikontribusikan oleh siapa saja:

- [ascendia-himarpl](https://github.com/himarplupi/ascendia-himarpl) - Situs web utama kabinet HIMARPL saat ini
- [blog-himarpl](https://github.com/himarplupi/blog-himarpl) - Platform blog staf HIMARPL
- [bot-himarpl](https://github.com/himarplupi/bot-himarpl) - Bot Telegram HIMARPL (Phoenix-chan)
- [pmb-himarpl](https://github.com/himarplupi/pmb-himarpl) - Portal Informasi untuk Mahasiswa Baru
- [mabim-himarpl](https://github.com/himarplupi/mabim-himarpl) - Situs Web Masa Bimbingan Rekayasa Perangkat Lunak Periode 2024
- [docs-himarpl](https://github.com/himarplupi/docs-himarpl) - Dokumentasi Teknis HIMARPL (situs web ini)
