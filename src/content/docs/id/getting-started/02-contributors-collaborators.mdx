Memahami peran kontributor dan kolaborator dalam proyek HIMARPL

## Apa itu Kontributor?

Kontributor adalah siapa saja yang berkontribusi pada proyek HIMARPL melalui:

- Membuat isu/PR (Pull Request)
- Mengomentari isu/PR
- Memperbaiki bug
- Menambahkan fitur
- Memperbarui dokumentasi
- Melaporkan isu
- Serta bentuk kontribusi lainnya

## Apa itu Kolaborator?

Kolaborator adalah anggota tim yang aktif berpartisipasi dalam diskusi dan pengambilan keputusan proyek dengan akses untuk:

- Meninjau isu/PR
- Mengelola isu/PR
- Melakukan tinjauan pada kode
- Menerima atau menolak Pull Request
- Memelihara repositori

Ini termasuk staf HIMARPL dan pemelihara web di Departemen Kominfo.
