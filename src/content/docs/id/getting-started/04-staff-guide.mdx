## Tanggung Jawab Utama Anda

Sebagai anggota staf di Kominfo, Anda akan bertanggung jawab untuk:

1. **Ketersediaan Layanan**
   - Memantau uptime situs web
   - Menangani masalah teknis dengan cepat
   - Memastikan semua layanan berjalan lancar

2. **Akses & Keamanan**
   - Mengelola tingkat akses pengguna
   - Menjaga kredensial tetap aman
   - Memantau masalah keamanan

3. **Pembaruan Reguler**
   - Melakukan pembaruan situs web
   - Memperbarui konten jika diperlukan
   - Memperbarui dependensi

4. **Koordinasi Pengembangan**
   - Bekerja dengan pengembang lain
   - Meninjau perubahan kode
   - Mengkoordinasikan perilisan fitur

5. **Dokumentasi**
   - Menjaga dokumentasi tetap terkini
   - Mendokumentasikan perubahan sistem
   - Mencatat keputusan teknis

## Prosedur Harian

### 1. Pemeliharaan Reguler
- Melakukan _backup_ mingguan

### 2. Manajemen Masalah

- Menanggapi laporan pengguna
- Mengkoordinasikan perbaikan _bug_

### 3. Komunikasi Tim
- Menghadiri rapat tim
- Memperbarui laporan kemajuan
- Berkoordinasi dengan departemen lain
- Mendokumentasikan keputusan

## Input Data Staf Baru

Ikuti langkah-langkah ini untuk menambahkan anggota staf baru:

1. **Akses Panel Admin**
   - Buka <a href="https://admin.himarpl.com" target="_blank" rel="noopener noreferrer">admin.himarpl.com</a>
   - _Login_ menggunakan akun utama HIMARPL

2. **Navigasi ke Bagian Staf**
   - Buka bagian `Staff`
   - Ubah periode ke tahun saat ini (misalnya, 2025)

:::note[CATATAN]
Jika tidak ada data untuk periode saat ini, Anda dapat menambahkannya secara manual menggunakan Prisma Studio.
:::

3. **Siapkan Gambar Staf**
   - Unggah foto staf ke repositori <a href="https://github.com/himarpl/assets-himarpl" target="_blank" rel="noopener noreferrer">assets-himarpl</a>
   - Konversi tautan GitHub ke tautan CDN menggunakan <a href="https://www.jsdelivr.com/github" target="_blank" rel="noopener noreferrer">jsdelivr</a>

4. **Tambahkan Anggota Staf**
   - Klik "Add New Staff"
   - Isi detail staf
   - Tempelkan tautan CDN jsdelivr di bidang `image`
   - Simpan perubahan

:::note[TIPS]
Anda dapat merujuk data staf periode sebelumnya untuk melihat contoh data entri yang diformat dengan benar.
:::
